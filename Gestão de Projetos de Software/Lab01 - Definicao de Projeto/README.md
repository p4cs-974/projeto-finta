# Lab 01 - Definição de Projeto

## Identificação do Projeto

**Nome do Projeto:** FINTA

**Descrição:** Aplicação web para centralizar e simplificar o acompanhamento de indicadores financeiros essenciais, incluindo ações, criptomoedas, taxa Selic e simulação de financiamentos.
