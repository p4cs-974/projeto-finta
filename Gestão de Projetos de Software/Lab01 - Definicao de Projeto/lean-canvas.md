# Lean Canvas - FINTA

## 1. Problemas

* Dificuldade em centralizar o acompanhamento de diferentes indicadores (Ações, Criptmoedas, Selic etc) em um só lugar.
* Complexidade para comparar taxas de financiamento de forma rápida.
* Falta de uma ferramenta simples para favoritar e monitorar variações de mercado sem excesso de informações irrelevantes.

## 2. Segmentos de Clientes

* Pessoas interessadas em acompanhar variações do mercado financeiro de forma prática.
* Investidores iniciantes ou entusiastas que buscam indicadores macroeconômicos (Taxa Selic).
* Usuários que buscam simular ou comparar condições de financiamento.

## 3. Propostas de Valor

* **Valor Único:** Centralização e simplicidade no acompanhamento de indicadores financeiros essenciais e pessoais.
* Diferente de home brokers complexos, o FINTA oferece uma interface limpa para análise rápida e comparação de taxas.

## 4. Soluções

* Módulo de acompanhamento de ações em tempo real.
* Ferramenta de visualização de taxas de financiamento.
* Sistema de autenticação para salvar preferências e "Favoritos".

## 5. Canais

* Aplicação Web (React) acessível via navegador.
* Possível divulgação em comunidades de desenvolvedores e finanças (ex: fóruns, GitHub, LinkedIn, Reddit).

## 6. Fontes de Receita

* Modelo *Freemium* (funcionalidades básicas gratuitas e análises avançadas pagas), como por exemplo: O assinante premium poderá receber notificações de queda de ações, taxas, selic entre outros para melhor analise e desempenho do mercado.
* Possíveis anuncíos no plano Free para segmentos mais utilizados, como criptmoedas por exemplo.

## 7. Estrutura de Custos

* Hospedagem do Backend (Node.js) e Frontend (React) para caso de grandes números de usúarios.
* Consumo de APIs de dados financeiros (ex: Yahoo Finance API, Alpha Vantage).
* Tempo de desenvolvimento e manutenção.

## 8. Métricas-Chave

* Número de usuários cadastrados (Autenticação).
* Quantidade de ações adicionadas aos "Favoritos".
* Frequência de uso da ferramenta de comparação de financiamento.
